# Homepage Mobile App
![image](https://github.com/user-attachments/assets/002bf89b-3bb7-474d-974d-b478de07eb5a)



1. Install dependencies

   ```bash
   yarn install
   ```

2. Start the app

   ```bash
   nvm use
   yarn run start
   ```

3. Start build ios and android

   ```bash
   nvm use    
   yarn run build
   ```
